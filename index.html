Text based Web RPG

In highschool I played quite a few different text based browser games including RPGs. I thought this would be a good initial project because I'd be able to find a place to use just about anything I learned in javascript.

The baseline project is now finished. It features a character creation and display, a shop and shopping cart feature, a navigation process, a basic quest system, a basic leveling system, and random encounter / combat mechanic. Everything is streamlined well enough that adding new content(areas, items, monsters, quests, etc.) is the work of minutes.

There is room for improvement here with a more involved leveling system, skills, classes, pictures, etc.

Navigation updated to a more general nsew using coordinates. Other systems such as save / load updated to account for this.

Equipment system added and fixed. Added health potions.
